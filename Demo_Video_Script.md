# Demo Video Script: FAAMG Financial Ratio Analyzer

## Video Duration: 2 minutes 30 seconds

---

### [0:00 - 0:15] Introduction

**On Screen:** Application interface with Streamlit branding

**Narration:**
"Hello everyone, welcome to this demo of the FAAMG Financial Ratio Analyzer. In the next two and a half minutes, I'll show you how this interactive tool can help you understand and compare the financial performance of major technology companies—Apple, Microsoft, Amazon, Google, and Meta—using core financial ratios that accountants and investors use every day."

---

### [0:15 - 0:45] Tool Overview and Navigation

**On Screen:** Streamlit sidebar with company and ratio selections

**Narration:**
"Let's start by exploring the sidebar. Here you can select which companies you want to analyze. I'm going to select Apple, Microsoft, and Amazon for this demo. Below that, you can choose which financial ratio you want to examine—Return on Equity, Net Profit Margin, Gross Profit Margin, or Asset Liability Ratio. Each ratio measures a different aspect of financial health, and I'll explain what each one means as we go."

---

### [0:45 - 1:15] Viewing Key Metrics

**On Screen:** Key Metrics Summary section with average, max, min, and latest values

**Narration:**
"Once you've selected your companies and ratio, the tool immediately calculates and displays key metrics. Here you can see the average value across all years, the maximum and minimum values, and the most recent 2025 figure. This gives you a quick sense of where each company stands. For example, looking at Return on Equity, you can immediately see which company is generating the most return relative to shareholder equity."

---

### [1:15 - 1:45] Interactive Visualization

**On Screen:** Line chart showing ROE trend over 2021-2025 for selected companies

**Narration:**
"The visualization section shows the trend over the five-year period from 2021 to 2025. You can toggle between a line chart, which is great for seeing trends over time, and a bar chart, which makes year-by-year comparisons easier. Notice how you can see the trajectory of each company's performance—some companies show improvement, while others may have temporary dips. These visual patterns help investors understand not just where a company is, but which direction it's heading."

---

### [1:45 - 2:15] Auto-Generated Insights

**On Screen:** Insight box with performance summary and key observations

**Narration:**
"One of the most helpful features is the auto-generated insights section. Based on your selected companies and ratio, the tool automatically produces a summary of performance, identifies the best and worst performers, and explains what the numbers mean in practical terms. This is especially useful for students learning financial analysis—you get professional-quality interpretation without needing years of experience. You can also expand the ratio definitions section to learn exactly what each metric means and how to interpret the values."

---

### [2:15 - 2:30] Closing and Call to Action

**On Screen:** Final view of the application with data download option

**Narration:**
"Finally, if you want to do your own analysis or include the data in a report, you can download everything as a CSV file with just one click. That's the FAAMG Financial Ratio Analyzer—interactive, educational, and practical. You can run this yourself by downloading the code and running 'streamlit run app.py' in your terminal. Thank you for watching, and happy analyzing!"

## Visual Cues for Screen Recording

### Before Recording:
1. Ensure browser is maximized for best view
2. Have all company tickers selected (AAPL, MSFT, AMZN, GOOGL, META)
3. Pre-select "ROE" as the default ratio view
4. Clear any cached data by clicking "Refresh Data" button

### Recording Tips:
- Move mouse slowly and deliberately when pointing to features
- Pause briefly on key areas to allow viewers to follow
- Use keyboard shortcuts for Streamlit (not needed, mouse is fine)
- Speak clearly at a measured pace

### Post-Recording Editing:
- Add subtle zoom effects on key metrics sections
- Consider adding caption overlays for the ratio formulas
- Background music should be minimal or absent
- Ensure audio levels are consistent throughout
